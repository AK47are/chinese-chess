# 说明
存放数据结构与算法课程的课程设计：可视化棋盘覆盖；可视化中国象棋。

使用 Qt C++来制作图形化交互，在 Qt Creator 中编写。

使用方法：在 Qt Creator 中点击对应项目的 CmakeLists.txt 即可打开项目。
